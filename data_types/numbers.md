Numbers are used to store numeric values in Python.

int is used for whole numbers

float is used for decimal numbers

complex is used for numbers with real and imaginary parts

Types of Numeric Data

| Type    | Description                    |
| ------- | ------------------------------ |
| int     | Stores whole numbers           |
| float   | Stores decimal values          |
| complex | Stores real + imaginary values |


a = 10        int (integer data type)
b = 3.5       float (floating-point data type)
c = 2 + 3j    complex (complex number data type)

print(a)
print(b)
print(c)
