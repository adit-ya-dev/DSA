Python mainly has numeric, text, boolean, and collection data types.

int – Stores whole numbers

float – Stores decimal numbers

complex – Stores numbers with real and imaginary parts

bool – Stores True or False values

str – Stores text or characters

list – Ordered and changeable collection of items

tuple – Ordered but unchangeable collection of items

set – Unordered collection of unique items

frozenset – Immutable version of a set

dict – Stores data as key–value pairs

range – Generates a sequence of numbers

bytes – Immutable binary data

bytearray – Mutable binary data

NoneType – Represents absence of a value (None)
